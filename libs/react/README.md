# @finance/react

Base React component library to be consumed by other libraries and applications.

Generated using the below command.
```bash
nx generate @nx/react:library react --publishable --importPath @finance/react
```

* `nx build react`: Builds this library.
* `nx build-storybook react`: Builds the storybook code so it can be deployed.
* `nx lint react`: Lints this library for style issues.
* `nx storybook react`: Runs the storybook application locally.
* `nx test react`: Runs this library's tests (add `--watch` to rerun on source changes).
